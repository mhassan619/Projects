# Ecommerce Price Tracker (Python)

## Overview
This project automates product data extraction from an ecommerce website and exports structured data into Excel format.

## Features
- Multi-page scraping
- Price categorization
- Timestamp logging
- Excel export
- Error handling

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas
- OpenPyXL

## Business Use Case
- Competitor price monitoring
- Market research
- Data automation
- Product catalog tracking

## How to Run
pip install -r requirements.txt
python scraper.py
