# Excel Automation & Report Generator

## Overview
Automates Excel reporting by cleaning messy sales data, generating summary statistics, top products, and monthly revenue.

## Features
- Remove duplicates & clean missing data
- Total sales and orders summary
- Top 5 products by revenue
- Monthly revenue report
- Multi-sheet Excel output

## Technologies
- Python
- Pandas
- OpenPyXL

## How to Run
pip install -r requirements.txt
python report_generator.py
