# AI Business Chatbot (Flask)

This is a simple **AI Business Chatbot** built using **Python Flask, HTML, CSS, and JavaScript**.  
The chatbot can answer common business questions such as services, pricing, delivery time, payment methods, and contact information.

## Features

- Simple AI keyword-based chatbot
- Built with Python Flask
- Clean ChatGPT-style UI
- Real-time chat using JavaScript fetch API
- Responsive chat interface

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
