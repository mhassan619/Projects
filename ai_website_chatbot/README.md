# AI Business Assistant 🤖

A simple AI-powered business support chatbot built using **Python (Flask)** and a **modern web chat interface**.  
This assistant helps businesses automate basic customer interactions and provide quick responses through an AI model.

---

## 🚀 Features

- Clean and modern chat interface
- Real-time AI responses
- Lightweight Flask backend
- Integration with local AI models (TinyLlama)
- Easy to customize for different business use cases

---

## 🛠 Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- TinyLlama (Local AI model via Ollama API)
