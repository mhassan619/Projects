# 🔐 Password Generator

A secure and customizable password generation utility built with Python.

## 🚀 Features
- **Customizable Length:** Generate passwords of any desired length.
- **Complexity Control:** Users can choose to include or exclude numbers and special characters.
- **Secure Randomization:** Uses Python's `random` module for unpredictable character selection.

## 💻 Tech Stack
- **Language:** Python
- **Modules:** `random`, `string`

## 📖 Usage
Run the script and follow the prompts:
```bash
python app.py
Type "yes" to include numbers/symbols or "No" to exclude them.
