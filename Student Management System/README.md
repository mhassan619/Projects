# 🎓 Advanced Student Management System

An enterprise-grade CLI application built to demonstrate modular software architecture and advanced Python programming principles.

## 🌟 Key Features
- **Modular Architecture:** Separate modules for Students, School Logic, and Utilities.
- **Custom Error Handling:** Implements specialized exception classes for robust data validation.
- **Decorators & Generators:** Uses decorators for logging/performance and generators for memory-efficient data streaming.
- **OOP Principles:** Deep implementation of Classes, Inheritance, and Encapsulation.

## 🛠️ Tech Stack
- **Language:** Python 3.10+
- **Concepts:** Object-Oriented Programming (OOP), Custom Decorators, Exception Handling, File I/O.
