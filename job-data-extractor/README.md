# Job Listings Scraper + Data Processor (Python)

## Overview
Scrapes job listings from a website (practice site) and exports clean CSV data. Demonstrates data extraction, cleaning, deduplication, and optional keyword filtering.

## Technologies
- Python
- Requests
- BeautifulSoup
- Pandas

## Features
- Multi-page scraping
- Duplicate removal
- Optional keyword filter
- CSV export

## How to Run
pip install -r requirements.txt
python job_scraper.py
