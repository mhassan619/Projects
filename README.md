## 💻 How to Clone & Run Projects Locally

Each project directory inside this workspace is standalone. To launch any script:

1. **Clone the Master Workspace:**
```bash
   git clone [https://github.com/mhassan619/Projects.git](https://github.com/mhassan619/Projects.git)
   cd Projects

```
 2. **Isolate an Execution Directory (e.g., PDF Merger):**
```bash
   cd "PDF Merger"

```
 3. **Execute the Underlying Engine:**
```bash
   python main.py  # Or target script name

```
